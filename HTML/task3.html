Semantic Tags

Semantic Tags are used to describe the content of a page. One Developer Create HTML code other Developer can understand.

1)Semantic Tags Example

<header> - Represents the header of a page

<nav> - Represents the navigation links of a page

<main> - Represents the main content of a page

<section> - Represents a standalone section of content

<article> - Represents a self-contained article

<aside> - Represents content aside from the main content

<footer> - Represents the footer of a page or section

2)Block Level Elements

Block level elements are elements that take up the full width available and start on a new line.

3)Block Level Elements Example

<div> - Represents a generic container for flow content

<h1> to <h6> - Represents headings of different levels

<p> - Represents a paragraph of text

4)Images

Images can be added using the <img> tag, which requires a source attribute to specify the image file.
