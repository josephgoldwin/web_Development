Image Tag Attributes

  • src: Specifies the path to the image.
  • alt: Provides alternative text for the image if it cannot be displayed.
  • width: Sets the width of the image.
  • height: Sets the height of the image.
  • title: Specifies extra information about the image (displayed as a tooltip).
  • loading: Specifies whether the image should be loaded immediately or lazily.
  • class: Specifies one or more class names for the image (for CSS styling).
  • id: Specifies a unique identifier for the image.
  • usemap: Associates the image with a client-side image map.
  • crossorigin: Specifies how the image should be fetched (e.g., anonymous or use credentials).
  • decoding: Specifies how the browser should decode the image (e.g., sync, async, or auto).
  • referrerpolicy: Specifies the referrer policy for the image.

Anchor Tag Attributes

  • href: Specifies the URL of the page the link goes to.(eg https://www.example.com),img/gold.jpg
  • target: Specifies where to open the linked document (default: _self)(e.g., _blank, _self, _parent, _top).
    1. _blank: Opens the link in a new tab or window.
    2. _self: Opens the link in the same frame as it was clicked (default).
    3. _parent: Opens the link in the parent frame.
    4. _top: Opens the link in the full body of the window.
  • download: Indicates that the link is for downloading a resource.(eg goldwin.jpg)
  • rel: Specifies the relationship between the current document and the linked document.(eg nofollow, noreferrer, noopener)
  • hreflang: Specifies the language of the linked document.(eg en-US, fr-FR, es-ES,ta,hi)
  • type: Specifies the MIME type of the linked document. (eg text/html, image/jpeg, application/pdf)
