2)What is an Element? and Types?

An element is the complete structure: 1)Opening tag, 2)Content 3)Closing tag

Block Elements - div, p, h1, h2, h3, h4, h5, h6, ul, ol, dl, form, table, tr, td, th, hr, br, hr, pre, script, style, noscript, iframe, article, section, header, footer, nav, aside, main, figure, figcaption, details, summary, menu, menuitem, command, keygen, progress, meter, canvas, output, audio, video, embed, source, track, param, object, applet

Inline Elements - a, abbr, acronym, address, b, bdo, big, cite, code, dfn, em, i, kbd, label, q, s, samp, small, span, strong, sub, sup, tt, var

Empty Elements - br, hr, img, input, link, meta, param, source, wbr

Non-Visible Elements - noscript, script, style

Nested Elements - div, p,goldwin,p,div

3)What is Tags?

A tag tells the browser how to display content.Tags are written in angle brackets

4)Tags - Explain

html-Root element

head-Contains meta-information about the document

body-Contains the content of the web page

title-Defines the title of the document

h1-h6-Defines headings of different levels

p-Defines a paragraph

div-Defines a division or section in the document

span-Defines a section in a document

ul-Defines an unordered list

ol-Defines an ordered list

li-Defines a list item

a-Defines a hyperlink

img-Defines an image

table-Defines a table

tr-Defines a table row

td-Defines a table cell

th-Defines a table header cell

form-Defines an HTML form for user input

input-Defines an input control

button-Defines a clickable button

label-Defines a label for an input element

select-Defines a drop-down list

option-Defines an option in a drop-down list

textarea-Defines a multi-line text input control

iframe-Defines an inline frame

script-Defines a client-side script

style-Defines style information for a document

link-Defines the relationship between a document and an external resource (most used to link to stylesheets)

meta-Defines metadata about an HTML document

noscript-Defines an alternate content for users that have disabled scripts in their browser

blockquote-Defines a section that is quoted from another source

cite-Defines the title of a work

code-Defines a piece of computer code

em-Defines emphasized text

strong-Defines important text

small-Defines smaller text

sub-Defines subscript text

sup-Defines superscript text

abbr-Defines an abbreviation or acronym

address-Defines contact information for the author/owner of a document

del-Defines text that has been deleted from a document

ins-Defines text that has been inserted into a document

q-Defines a short inline quotation

var-Defines a variable

kbd-Defines keyboard input

time-Defines a specific time (or range of time)

progress-Defines the progress of a task

meter-Defines a scalar measurement within a known range (a gauge)

details-Defines additional details that the user can view or hide

summary-Defines a summary, caption, or legend for a details element

dialog-Defines a dialog box or other interactive component

template-Defines a template for content that is not to be rendered immediately when the page loads

canvas-Defines a drawing area for graphics

svg-Defines a container for SVG graphics

math-Defines a mathematical expression

object-Defines an embedded object

embed-Defines an embedded content, such as a multimedia file

source-Defines multiple media resources for elements like audio and video

track-Defines text tracks for elements like video and audio

wbr-Defines a possible line break
